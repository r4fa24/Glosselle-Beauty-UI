# Glosselle - Luxury Lip Gloss Storefront 💄✨
A high-end, responsive e-commerce landing page designed for premium beauty branding.

## 🚀 Overview
Glosselle is a front-end development project focused on delivering a luxury user experience (UX) for beauty retail. The project emphasizes clean typography, smooth navigation, and a modern aesthetic to drive customer engagement and brand loyalty.

## ✨ Key Features
* **Luxury UI/UX Design:** Implemented a sophisticated visual hierarchy using high-end typography (Abril Fatface, Cinzel Decorative) to establish premium brand identity.
* **Fully Responsive Layout:** Utilized flexible CSS containers to ensure the storefront remains elegant and functional across all screen sizes.
* **Feature-Driven Architecture:** Organized with modular sections including unique selling points (USPs), customer testimonials, and clear call-to-action (CTA) gateways.
* **Performance Optimization:** Integrated optimized web fonts and localized asset management for fast loading times.

## 🛠️ Tech Stack
* **Front-end:** HTML5, CSS3
* **Design:** Custom Google Fonts integration
